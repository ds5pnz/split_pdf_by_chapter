# PDF Chapter Splitter (PDF 챕터별 분할기)

이 프로그램은 PDF 파일의 목차(Table of Contents)를 분석하여 최상위 챕터(Level 1)별로 PDF를 자동 분할해 주는 도구입니다. 사용자 친화적인 GUI 인터페이스와 명령행(CLI) 방식을 모두 지원합니다.

## 주요 기능

*   **자동 챕터 인식**: PDF 내부의 목차 정보를 읽어 챕터별 시작과 끝 페이지를 자동으로 계산합니다.
*   **GUI 모드**: `tkinter` 기반의 직관적인 화면을 통해 파일을 선택하고 진행 상황을 실시간으로 확인할 수 있습니다.
*   **CLI 모드**: 명령행 인자를 통해 빠르게 작업을 수행할 수 있어 기존 워크플로우에 통합하기 좋습니다.
*   **자동 파일 정리**: 분할된 파일은 원본 파일명을 딴 별도의 디렉토리에 깔끔하게 저장됩니다.
*   **파일명 정리**: 챕터 제목의 특수문자를 제거하여 안전한 파일명을 생성합니다.

## 요구 사항

*   **Python**: 3.x 버전 이상
*   **필수 라이브러리**:
    *   `PyMuPDF` (fitz): PDF 처리 및 목차 분석용

## 설치 방법

필요한 라이브러리를 설치합니다.

```bash
pip install pymupdf
```

## 사용 방법

### 1. GUI 모드 (기본)
아무런 인자 없이 프로그램을 실행하면 GUI 창이 나타납니다.

```bash
python main.py
```
1. 'PDF 파일 선택' 버튼을 눌러 분할할 파일을 선택합니다.
2. '분할 시작' 버튼을 클릭합니다.
3. 원본 파일과 같은 위치에 생성된 폴더 내에서 분할된 PDF 파일들을 확인할 수 있습니다.

### 2. CLI 모드
명령행에서 직접 파일 경로를 전달하여 실행할 수 있습니다.

```bash
python main.py "path/to/your/document.pdf"
```

## 저장 방식
예를 들어 `Python_Guide.pdf` 파일을 분할하면 다음과 같이 구성됩니다.
*   **저장 위치**: `Python_Guide/` (원본 파일과 같은 디렉토리 내 생성)
*   **파일명 형식**: `Python_Guide - [챕터 제목].pdf`

## 주의 사항
*   PDF 내부에 목차(TOC) 정보가 포함되어 있어야 정상적으로 작동합니다.
*   최상위 수준(Level 1)의 목차를 기준으로 분할을 수행합니다.
